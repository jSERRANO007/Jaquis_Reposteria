# reporteria.md

Proyecto para Claude Code: construir y evolucionar el sistema JAQUIS Repostería Gourmet con énfasis en UX premium, catálogo visual, pedidos por web y apoyo por WhatsApp, reportes, pantalla tipo cocina, auditoría y mantenibilidad.

## Requerimientos clave
- catálogo con imágenes administrables
- pedidos con adelanto del 50% cuando cantidad >= 10
- reportes por producto, categoría y fecha
- dashboard y drill-down en fases siguientes
- buenas prácticas de desarrollo, QA, trazabilidad e ITIL
